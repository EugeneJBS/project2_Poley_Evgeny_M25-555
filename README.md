# Простая консольная база данных

Учебный проект по реализации легковесной консольной "базы данных", использующей JSON-файлы в качестве хранилища. Система поддерживает базовые операции CRUD, создание/удаление таблиц и фильтрацию данных.

Проект предназначен для практики работы с Python, управления зависимостями через Poetry, автоматизации задач через Makefile и контроля качества кода с помощью Ruff.

## Установка

Все зависимости устанавливаются одной командой через систему автоматизации:

```bash
make install
```

## Запуск программы

Запуск интерактивной консоли базы данных:

```bash
make project
```

После старта приложения необходимо ввести имя пользователя для начала сессии, после чего станет доступен ввод команд.

## Основные команды

### Управление таблицами
* `create_table <имя> <колонка:тип> ...` — создание новой таблицы с типизацией (поддерживаются `str`, `int`, `bool`).
* `list_tables` — вывод списка всех существующих таблиц.
* `drop_table <имя>` — полное удаление таблицы и связанных данных.
* `info <таблица>` — просмотр структуры колонок и типов данных таблицы.

### Работа с данными
* `insert into <таблица> values (значения...)` — добавление новой записи.
* `select from <таблица>` — вывод всех записей таблицы.
* `select from <таблица> where <поле> = <значение>` — выборка с фильтрацией.
* `update <таблица> set <поле> = <новое> where <поле> = <условие>` — обновление данных.
* `delete from <таблица> where <поле> = <значение>` — удаление конкретных записей.
* `help` — вызов справки по командам.
* `exit` — завершение работы.

## Пример использования

Последовательность действий для создания базы пользователей и управления ими:

```text
create_table users name:str age:int is_active:bool
insert into users values ("Sergei", 28, true)
select from users
update users set age = 29 where name = "Sergei"
delete from users where ID = 1
drop_table users
```

## Проверка качества кода

Для проверки соответствия кода стандартам PEP8 и поиска ошибок используется линтер Ruff:

```bash
make lint
```

## Сборка и установка пакета

Собрать дистрибутив пакета (wheel и sdist):
```bash
make build
```

Тестовая публикация (без отправки в основной PyPI):
```bash
make publish
```

Установка собранного пакета в систему:
```bash
make package-install
```
